# Калькулятор стоимости сауны

## Обзор

Интерактивный калькулятор для расчета стоимости посещения сауны на основе:
- Типа услуги (LUX, На дровах/Хамам, Микс)
- Периода времени (Будни/Выходные)
- Количества гостей (взрослые, дети)
- Дополнительных услуг (банщик, веники)

## Структура файлов

- **index.html** - HTML разметка калькулятора (строки 426-545)
- **js/calculator.js** - JavaScript логика калькулятора (5.6 KB)
- **css/styles.css** - CSS стили калькулятора (добавлено ~250 строк в конец)

## Основные компоненты

### 1. Выбор типа услуги
- LUX Sauna
- На дровах или Хамам
- Русская сауна + Хамам (+30%)

### 2. Выбор периода
- Будни: Четверг (весь день), Пятница (до 16:00)
- Выходные: Пятница (после 16:00), Суббота, Воскресенье

### 3. Количество гостей
- Взрослые (12+) - оплачиваются полностью
- Дети (6-12 лет) - 50% скидка (6.25€ за человека)
- Дети (до 6 лет) - бесплатно

### 4. Дополнительные услуги
- Банщик: 12.50€ за заход
- Березовый веник: 9.00€ за штуку
- Дубовый веник: 10.00€ за штуку

## Правила расчета

1. **Минимальный чек**: 2 человека (если выбрано меньше, расчет идет за 2)
2. **Экстра взрослые**: При количестве > 10 человек, каждый дополнительный взрослый стоит 12.50€
3. **Детский тариф**:
   - До 6 лет: бесплатно
   - 6-12 лет: 50% от тарифа дополнительного места (6.25€)
4. **Дополнительные услуги**: Складываются к базовой цене

### Примеры расчета

**Пример 1**: 2 взрослых, LUX, будни
- Базовая цена: 25€

**Пример 2**: 5 взрослых + 1 ребенок (6-12), LUX, выходные
- Базовая цена: 50€ (за 5 взрослых)
- Дети 6-12: 6.25€
- **Итого: 56.25€**

**Пример 3**: 12 взрослых, На дровах, будни
- Базовая цена: 75€ (за 10 человек)
- Экстра взрослые: 2 × 12.50€ = 25€
- **Итого: 100€**

## Технические детали

**JavaScript**:
- Чистый ванильный JavaScript без зависимостей
- Использует прямую таблицу цен для быстрых расчетов
- Event listeners для всех input элементов
- Форматирование цен в евро с 2 знаками после запятой

**CSS**:
- Гридовая система для формы (grid-template-columns)
- Адаптивные брейкпоинты (768px, 480px)
- Переходы и hover эффекты
- Темные стили для большей читаемости
