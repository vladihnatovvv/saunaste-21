# Реализация калькулятора стоимости сауны

## ✅ Выполненные задачи

### 1. HTML структура (index.html)
- ✓ Создана полная форма с 4 основными группами:
  - Выбор типа услуги (3 варианта: LUX, Дрова/Хамам, Микс)
  - Выбор периода (Будни/Выходные с подсказками времени)
  - Счетчики для гостей (Взр, Дети 6-12, Дети до 6)
  - Дополнительные услуги (Банщик, Веники)
- ✓ Динамическая таблица расходов с разбивкой по категориям
- ✓ Кнопка "Забронировать" с интеграцией в форму обратного звонка
- ✓ Ссылка на калькулятор в боковом меню

### 2. JavaScript логика (js/calculator.js)
- ✓ Таблица цен с полной тарифной сеткой
  - 3 типа услуг (LUX, Дрова, Микс)
  - 2 типа периода (Будни, Выходные)
  - 9 тарифных точек (2-10 человек)
- ✓ Алгоритм расчета:
  - Минимальный чек 2 человека
  - Экстра взрослые по 12.50€
  - Дети 6-12 по 6.25€ (50% скидка)
  - Дополнительные услуги
- ✓ Динамическое обновление при любом изменении
- ✓ Форматирование цен в формате "XX.XX €"
- ✓ Условное отображение строк стоимости

### 3. CSS стили (css/styles.css)
- ✓ Полностью стилизованная форма (250+ строк)
- ✓ Радио-кнопки с hover эффектами
- ✓ Ввод чисел с фокусом и переходами
- ✓ Таблица стоимости с разделителями
- ✓ Адаптивный дизайн:
  - Desktop (Full width)
  - Tablet (768px) - сужение формы
  - Mobile (480px) - однолонная сетка
- ✓ Соответствие дизайну сайта:
  - Цвета: #AB8C6B (основной), #1a1a1a (текст)
  - Шрифты: Inter (основной), Montserrat (заголовки)
  - Округление: 16px border-radius

### 4. Функциональность
- ✓ Real-time расчет без задержек
- ✓ Валидация (no negative numbers с min="0")
- ✓ Подсказки для временных интервалов
- ✓ Hint для бесплатных детей
- ✓ Полная разбивка стоимости
- ✓ Интеграция с системой бронирования
